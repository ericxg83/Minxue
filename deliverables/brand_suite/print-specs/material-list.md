# 敏学空间品牌宣传落地清单

## 品牌定位

- 品牌名：敏学成长中心
- 核心气质：温暖、专业、清晰、可信
- 核心表达：看见问题，陪着进步

## 空间信息

- 总面积：约 40 平方米
- 墙高：3 米
- 前台区域：4m x 1.5m
- 教室 A：3m x 4m
- 教室 B：4m x 4m
- 过道：1m x 3m

## 物料清单

1. 前台主视觉墙
- 建议尺寸：宽 4m x 高 2.6m 到 2.8m
- 核心文案：敏学成长中心 / 看见问题，陪着进步
- 辅助文案：错题整理 / 学情诊断 / 练习追踪 / 成长反馈

2. 前台服务介绍立牌
- 建议尺寸：A3 立牌或 60cm x 90cm
- 标题：我们怎么帮助孩子提升
- 内容：发现问题 / 整理问题 / 跟踪提升

3. 过道三联文化墙
- 建议尺寸：每幅 60cm x 120cm，共 3 幅
- 主题：发现 / 整理 / 提升

4. 教室 A 方法主题墙
- 建议尺寸：主墙 2.8m x 2m 左右
- 标题：会整理，才会进步

5. 教室 B 成长主题墙
- 建议尺寸：主墙 3m x 2m 左右
- 标题：每一次进步，都值得被记录

6. 招生海报
- 建议尺寸：A1 / A2
- 标题：让孩子的每一次练习，都更有结果

7. 家长单页
- 建议尺寸：A4 双面或三折页
